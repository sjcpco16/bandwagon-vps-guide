## 第二步：注册账号 + 完成购买（图文步骤）

### 注册注意事项

打开搬瓦工官网后，点右上角 **Register**。填写信息时有几个坑要避：

- 邮箱必须填真实的，购买完成后服务器信息和 SSH 端口都发到这个邮箱
- 手机号可以不填真实的，但 Country 选项必须和你实际所在地区一致
- 不要开着代理注册，容易被系统判定为欺诈订单，直接被拒单（自动退款）

### 购买流程（编号步骤）

1. 进入对应套餐页面，选择付款周期（推荐选 Annually 年付，单价最低）
2. 选择机房（对国内用户来说，选 Los Angeles DC6 或 DC9 是最常见的起点）
3. 点击 **Add to Cart**，进入结账页面
4. 如果有官方促销码，在 Promotional Code 框里填入，点 Validate Code 验证
5. 选择付款方式：**Alipay（支付宝）** 或 **Unionpay（银联）** 是最方便的，也支持 PayPal 和信用卡
6. 勾选同意服务条款，点 **Complete Order** → 跳转付款页面扫码完成

付款完成后系统会发一封邮件到你注册的邮箱。这封邮件很重要，里面有：VPS 的 IP 地址和 SSH 端口号，得留着备用。

## 第三步：进入 KiwiVM 后台管理面板

购买成功后，登录账号，依次点击 **Services → My Services**，找到你的套餐，点 **Manage**，然后点 **Open KiwiVM**，就进后台了。

KiwiVM 是搬瓦工自家开发的管理面板，不需要额外安装，直接网页访问。

### KiwiVM 主要功能图解

**Main Controls（主控制台）**：显示你的 VPS 基本信息

- IP Address：服务器公网 IP
- SSH Port：连接用的端口（搬瓦工默认不是 22，是个随机的四五位数端口，邮件里也有）
- Status：服务器当前是否在运行
- 可以在这里执行关机、重启、强制重启操作

**Install New OS（重装系统）**：

- 支持 Debian、Ubuntu、CentOS 等主流 Linux 发行版，还有几个带 BBR 加速版本
- 重装之前必须先关机（Stop）
- 勾选确认提示后点击安装，系统会随机生成一个新 root 密码，重装完邮件通知

**Migrate to another DC（迁移机房）**：

- CN2 GIA-E 套餐在这里能切换全部可用机房，免费操作
- 迁移后 IP 会变，数据不丢失，但建议迁移前先备份重要文件

**Backups（自动备份）**：

- 搬瓦工每六天自动快照一次，完全免费
- 在 Snapshots 里可以随时还原，每台机器最多保留 2 个快照
- 想长期保留某个快照，点 **Set Sticky** 标记

## 第四步：SSH 连接服务器

这是很多新手第一个卡住的地方。

### SSH 连接所需信息

从 KiwiVM 主控台或者购买邮件里找到这三样东西：
- IP 地址
- SSH 端口号（不是 22，是那个几千位的随机端口）
- root 密码（初始密码在邮件里，也可以在 KiwiVM 主控台找到重置密码按钮）

### Windows 用户：用 Xshell 或 PuTTY

PuTTY 是最轻量的选择：

1. Host Name 填你的 IP 地址
2. Port 填你的 SSH 端口号
3. 保存 Session 名称方便下次直接 Load
4. 点 Open，login as 填 `root`，回车
5. 输入密码（注意：输密码时屏幕没有显示，这是正常现象，输完直接回车）

### Mac / Linux 用户：直接用终端

打开终端，输入：

bash
ssh root@你的IP地址 -p 你的端口号


按提示输入密码，回车登录。第一次连接会提示 Are you sure you want to continue connecting，输入 `yes` 确认就行。

登录成功后，命令行前面显示 `root@`... 就说明连上了。

## 第五步：搬瓦工 VPS 常见搭建用途教程

### 用途一：搭建网站（LNMP 建站）

LNMP 是 Linux + Nginx + MySQL + PHP 的组合，适合跑 WordPress、个人博客等。

登录服务器后，安装 LNMP 一键包：

bash
wget http://soft.vpser.net/lnmp/lnmp2.0.tar.gz -cO lnmp2.0.tar.gz && tar zxf lnmp2.0.tar.gz && cd lnmp2.0 && ./install.sh lnmp


安装过程中会让你选 MySQL 版本和 PHP 版本，新手选默认推荐的就行，整个过程大概 20-40 分钟。装完就能在 KiwiVM 里看到服务器跑起来了。

### 用途二：部署私有化 AI 应用（Dify / n8n）

这是搬瓦工近两年比较热门的用法。

Dify 是目前最成熟的开源 AI 应用平台之一，在搬瓦工上部署只需要 Docker：

bash
# 安装 Docker
curl -fsSL https://get.docker.com | sh
systemctl enable docker && systemctl start docker

# 拉取并启动 Dify
git clone https://github.com/langgenius/dify.git
cd dify/docker
docker compose up -d


部署 Dify 建议选 CN2 GIA-E 套餐中 2GB 内存那档，跑起来比较流畅，1GB 的在接入多个大模型时会有点吃力。

### 用途三：自建代理节点（V2Ray）

这是搬瓦工用户量最大的用途之一。

搬瓦工 CN2 GIA 线路的特点：电信和联通双程走 CN2 GIA 高端线路，移动走 CMIN2，晚高峰的抗拥堵能力比 CN2 GT 和普通国际线路强得多。

V2Ray 一键脚本可以大幅简化安装流程：

bash
bash <(curl -s -L https://git.io/v2ray.sh)


按提示选择协议（VMess 或 VLESS 都可以），设置端口，脚本会自动配置好。安装完会输出客户端需要的配置信息，截图保存备用。

客户端这边，iOS 用 Shadowrocket，Android 用 v2rayNG，Windows/Mac 用 v2rayN 或 Clash，把服务器生成的配置填进去就能用。

## 搬瓦工使用中的常见问题（FAQ）

**Q：搬瓦工支持哪些支付方式？**

A：支持支付宝（Alipay）、银联（Unionpay）、PayPal、信用卡，对国内用户最方便的是支付宝，直接扫码付款。

**Q：买了之后不满意可以退款吗？**

A：可以。30 天内不满意可以申请全额退款，在 KiwiVM 后台提交退款申请，系统自动处理。有朋友买了试了将近一个月觉得不合适，退款确实打回来了。

**Q：搬瓦工套餐可以换机房吗？换了 IP 会变吗？**

A：CN2 GIA-E 套餐和部分其他套餐支持在 KiwiVM 后台一键免费迁机房，迁移后 IP 会变，但数据不丢失。KVM 基础套餐只有美国弗里蒙特一个机房，不支持迁移。

**Q：购买后多久可以用？**

A：付款完成后几分钟内系统就会激活，登录邮箱查看服务器信息邮件，然后就能 SSH 登录了。

**Q：搬瓦工有没有现成的优惠码？**

A：搬瓦工会不定期发布官方折扣码，优惠码是循环折扣——买的时候用上了，续费也自动享受同等折扣。目前可以到搬瓦工官网活动页查看是否有当前有效的促销码。有码一定要在下单时填进去，省的是长期续费的钱。

**Q：KVM 套餐和 CN2 GIA-E 套餐差在哪里？**

A：核心差别在线路。KVM 套餐走的是美国普通线路，国内访问速度一般；CN2 GIA-E 套餐走 CN2 GIA 优化线路，电信联通到洛杉矶 DC6/DC9 延迟和速度明显更好，晚高峰也比较稳。如果对速度有要求，多花这点钱是值的。
